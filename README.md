# lsj170
このリポジトリは、日本言語学会170回大会(2025/6/28@明海大学)の口頭発表「南琉球宮古語伊良部島方言における再帰代名詞を用いた物語の1人称用法」（C-35）の
スライド資料とサンプルデータを格納したものです。
- lsj170_shimoji.pdf (スライド資料）
- sampletexts.pdf
- audioフォルダ

## 使用法・ダウンロード法
メインページのCode（緑ボタン）からzip形式で一括ダウンロードできます。
